# Code of Conduct

Linka is an open-source project. Everyone participating in issues, pull requests, and discussions is expected to be respectful and constructive.

## Expected Behavior

- Be welcoming and patient.
- Give feedback that is specific and useful.
- Respect different experience levels and perspectives.
- Keep discussions focused on the project.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Discriminatory language or behavior.
- Repeated off-topic or disruptive comments.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove comments, close issues, or block participation when behavior is harmful to the project or community.

If you see a problem, report it through the project issue tracker or contact the maintainer directly.
